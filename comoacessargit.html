==> PARA CONFIGURACAO GIT CONFIG --GLOBAL USER.NAME "DEFINIR NOME DO USUARIO"
==> GIT CONFIG --GLOBAL USER.EMAIL "DEFINIR E-MAIL USUARIO"
==> GIT CONFIG --GLOBAL CORE.EDITOR "DEFINIR EDITTOR DE CODIGO"
==> USE GIT STATUS (PARA VER OS ARQUIVOS QUE ESTÃO COMITADOS OU NECESSITAM REALIZAR A COMITAÇÃO)
==> PARA ADICIONAR OS ARQUIVOS BASTA COLOCAR " GIT ADD (NONE DO DOCUMENTO)
==> PARA COMITAR OS ARQUIVOS COLOCAR GIT COMMIT -m ("TEXTO ASSUNTO MODDIFICAÇÃO")
==> PARA VER OS ARQUIVOS COMITADOS GIT LOG
==> PARA REMOVER ARQUIVO GIT RESET --HARD "INSERIR TODO COMMIT EX(commit 8ba3a74adb9215d672c11ae1214982b0d58dec8e)"
PARA ARRUMAR O ARQUIVO SEM PERDER AS MUDANÇAS GIT RESER --SOFT (VOLTA PARA SER COMITADO) E GIT RESET --MIXERD (VOLTA PARA SER ADD) ==> CRIA BRANCH ( GIT BRANC NOME DO NOVO BRANCH)
